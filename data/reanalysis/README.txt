Add here all the reanalysis files in .grib or .nc format
This directory must contain one folder for each different year of data named "y_YYYY" where YYYY is the year.
The files inside the yearly folders must have the following naming structure: "YYYY_level_variable"
For example:

- /reanalysis
   - /y_2014
       - 2014_850_TEMPERATURE.nc
       - 2014_SURF_PCP.grib
   - /y_2015
       - 2015_850_TEMPERATURE.nc
       - 2015_SURF_PCP.grib
